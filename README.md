# Auth Example Project

## Project example of an auth including authorization and authentication use vite as frontend and express as backend.

This project included 2 main folders: `frontend` and `backend`

Run 2 app process:

frontend: `pnpm run dev`

backend: `pnpm run dev`

`Dont forget to set .env port in backend!`

## Main App

Fully documented code for backend and frontend can be accessed in:

frontend: `src\App.tsx`

backend: `src\index.ts`
